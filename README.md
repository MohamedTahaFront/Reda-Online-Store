# 🛒 Reda | Online Store  
### E-commerce Front-End Project

A modern, fully responsive e-commerce front-end application built to deliver a smooth and intuitive shopping experience. The project emphasizes clean UI/UX, efficient JavaScript logic, and dynamic data handling.

🔗 **Live Demo:**  
👉 https://mohamedtahafront.github.io/Reda-Online-Store/

---

## 🚀 Tech Stack

- **HTML5 & CSS3**  
  Semantic structure with responsive and modern design.

- **JavaScript (ES6+)**  
  Handles DOM manipulation, Fetch API, and application logic.

- **JSON**  
  Acts as a local database for dynamic product rendering.

- **Swiper JS**  
  Enables high-performance sliders and touch interactions.

- **Font Awesome**  
  Provides scalable vector icons.

---

## ✨ Features

### 🛍️ Product System
- Dynamic product rendering from `products.json`
- Automatic categorization (Mobiles, Electronics, Appliances, etc.)

### 🛒 Cart System
- Add to cart with duplicate prevention  
- Real-time quantity control (increase / decrease)  
- Dynamic subtotal & total calculation  
- Smart "Add to Cart" button state updates  

### 💾 State Management
- Cart data stored in `localStorage`
- Data persists after page refresh

### 💳 Checkout Experience
- Dedicated checkout page  
- Order summary with shipping calculation  
- Coupon code handling  

### ⚡ Performance Optimization
- Implemented **Event Delegation** for better performance and reduced memory usage

---

## 🛠️ How to Run the Project

1. Download or clone the repository  
2. Open the project using a Live Server (e.g., VS Code Live Server extension)  
3. Start exploring the store 🚀  

> ⚠️ Note: Using Live Server is required to ensure the `fetch()` API works correctly with the local JSON file.
